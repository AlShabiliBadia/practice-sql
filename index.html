Help & Examples

Getting Started:

  • Type SQL commands into the input box (the "shell").
  • Click the "Run SQL" button or press Ctrl+Enteror Cmd+Enter (Mac) to execute.
  • The results of your commands (or any errors) will appear in the "Output" area.
  • If a command runs successfully, the input box will clear automatically.

Pinning Output:

  • After running a command (like a SELECT query), click the "Pin Output" button next to the Output title.
  • A copy of the current output will appear in the "Pinned Outputs" section below.
  • This is useful for keeping table data or results visible while you run other commands.
  • Click the red 'X' on a pinned item to remove it.

Example Workflow:

  1. Create a table:
    CREATE TABLE products (id INTEGER PRIMARY KEY, name TEXT NOT NULL, price REAL);
  2. Insert data:
    INSERT INTO products (name, price) VALUES ('Laptop', 1200.50), ('Mouse', 25.00);
  3. View data:
    SELECT * FROM products;
  4. Click "Pin Output" to keep the product list visible.
  5. Query specific data:
    SELECT name, price FROM products WHERE price < 100;
